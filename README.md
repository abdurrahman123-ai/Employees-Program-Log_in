# 👥 Employees Log-in System

## 📋 Overview
A secure desktop application built for managing employee access. This project demonstrates database connectivity and user authentication logic.

## 🚀 Features
* Secure Login: Authentication with hashed passwords (optional).
* Database Integration: Stores employee credentials safely.
* User-friendly GUI: Clean interface for daily use.

## 🛠️ Tech Stack
* Language: Python
* Database: SQL / SQLite
* Library: Tkinter (for GUI)
